MCO vs RHCOS for systemd Issues

  • RHCOS: Owns systemd package and OS-level vulnerabilities
  • MCO: Manages systemd configuration and service files
  • CVE-2025-4598: Race condition in systemd-coredump = package issue = RHCOS

OpenShift Build Infrastructure

  • Build Component: Owns all container building infrastructure

  • CVE-2025-22871: net/http vulnerability in builder = Build issue
